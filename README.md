# GREEN GLASS - تطبيق إعادة تدوير الزجاج

## نظرة عامة
GREEN GLASS هو منصة شاملة لإدارة جمع وإعادة تدوير الزجاج في سيدي بلعباس، الجزائر.

## المميزات الرئيسية
- 🗺️ خريطة تفاعلية لنقاط التجميع
- 📱 طلبات جمع الزجاج للأفراد والمؤسسات
- 🎁 نظام مكافآت ونقاط
- 📊 إحصائيات بيئية
- 💬 نظام مراسلة داخلي
- 👨‍💼 لوحة تحكم إدارية

## التثبيت

### المتطلبات
- Node.js 18+
- MongoDB
- Google Maps API Key

### خطوات التثبيت

1. تثبيت التبعيات:
```bash
npm run install-all
```

2. إعداد ملف البيئة:
```bash
cp .env.example .env
```
قم بتعديل `.env` وأضف مفاتيح API الخاصة بك.

3. تشغيل المشروع:
```bash
npm run dev
```

- Backend: http://localhost:5000
- Frontend: http://localhost:5173

## البنية
- `backend/` - خادم Express وAPI
- `frontend/` - تطبيق React

## الأدوار
- **User**: يمكنه إنشاء طلبات جمع
- **Collector**: يمكنه قبول طلبات الجمع
- **Admin**: إدارة كاملة للنظام
